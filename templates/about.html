about this project

Mumbai House Price Predictor

A machine learning web application that predicts house prices across Mumbai using a Random Forest model trained on 76,000+ real listings.

the model

We use a Random Forest Regressor — an ensemble of 150 decision trees that vote together to predict price. It captures complex non-linear relationships between location, size, and price that simpler models like Linear Regression miss.

the dataset

Trained on 76,038 Mumbai listings covering 228 regions. Prices were normalised to Lakhs (₹), outliers removed, and categorical features encoded using One Hot Encoding before training.

RMSE ₹59.70 L
